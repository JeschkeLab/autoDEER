Contributing Guide
==================


There are three main ways to contribute to the project:

1. Submit a bug report: 
    * Go to the github issues page and click the "New Issue" button.
    * Fill out the form and submit it.
    * We will try and fix it as soon as possible.

2. Submit a feature request:
    * Go to the github discussion page and click the "New Discussion" button.
    * Fill out the form and submit it.
    * We will try and get back to you as soon as possible.

3. Join the development:
    * Fork the project.
    * Make your changes.
    * Submit a pull request.
